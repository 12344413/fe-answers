2023-2024 第一学期 考试试题 (A卷) 答案详解

📝 第1题（15分）
假定你现在持有的付红利股票的价格为150。计划在250天后签订一份该股票的远期合约多头。 无风险利率为5.25%（连续复利）。250天内股票按以下方式支付红利：第30天$1.25，第120天$1.25，第210天$1.25。 (1) 该远期合约的理论价格是多少？ (2) 若100天后股票价格为115，求此时该远期合约的价值。 (3) 到期时股票价格为130，求此时远期合约的价值。
✅ 第1题 解答
(1) 远期价格 步骤1：计算红利现值 I I = 1.25×e-0.0525×30/365 + 1.25×e-0.0525×120/365 + 1.25×e-0.0525×210/365 = 1.25×0.9957 + 1.25×0.9829 + 1.25×0.9702 = 1.2446 + 1.2286 + 1.2128 = 3.6860 步骤2：远期价格公式 F = (S - I)erT F = (150 - 3.6860)×e0.0525×250/365 = 146.3140×e0.03596 = 146.3140×1.0366 = 151.67
答：远期合约理论价格为 $151.67
(2) 100天后合约价值 100天后，还剩150天到期，剩余两次红利：第120天$1.25，第210天$1.25 剩余红利的现值： I' = 1.25×e-0.0525×20/365 + 1.25×e-0.0525×110/365 = 1.25×0.9971 + 1.25×0.9842 = 1.2464 + 1.2303 = 2.4767 远期合约多头价值： f = S - I' - K e-r(T-t) = 115 - 2.4767 - 151.67×e-0.0525×150/365 = 112.5233 - 151.67×0.9787 = 112.5233 - 148.43 = -35.91
答：多头头寸价值 -$35.91（即亏损$35.91）
(3) 到期时合约价值 到期时，远期价格已无时间价值，合约价值为： fT = ST - K = 130 - 151.67 = -21.67
答：到期时多头亏损 $21.67
📝 第2题（10分）
一个看跌期权标的资产为不分红的股票，有效期120天（按365天计），当前股价$47，无风险利率5%（连续复利），执行价格$50。问美式看跌期权和欧式看跌期权的价格下限各为多少？
✅ 第2题 解答
欧式看跌期权下限： p ≥ max(Xe-rT - S, 0) Xe-rT - S = 50×e-0.05×120/365 - 47 = 50×0.9837 - 47 = 49.185 - 47 = 2.185
欧式看跌期权下限 ≥ $2.19
美式看跌期权下限： 美式期权可以提前执行，下限更高： P ≥ max(X - S, 0) X - S = 50 - 47 = 3
美式看跌期权下限 ≥ $3.00
解释：美式期权可以立即以$50执行，赚取$3差价； 而欧式期权必须等到120天后，因此只能以折现值计算，下限更低。
📝 第3题（15分）
一家金融机构与公司X签订了一份利率互换协议。根据协议，机构收取10%固定利率，支付6个月LIBOR，名义本金$1000万，5年期每6个月支付一次。 假设公司X在第3年末违约，当时6个月期LIBOR为9%，对所有期限利率都为8%（连续复利）。求金融机构的损失。
✅ 第3题 解答
第3年末当期损失： 机构应收固定利息 = $1000万 × 6/12 × 10% = $500,000 机构应付浮动利息 = $1000万 × 6/12 × 9% = $450,000 当期净损失 = $500,000 - $450,000 = $50,000 剩余期限损失： 第3年到第5年剩余2年，共4个半年期。 假设远期利率均实现（=8%），则： 每期应收固定利息 = $500,000 每期应付浮动利息(LIBOR=8%) = $1000万 × 6/12 × 8% = $400,000 每期净收入 = $500,000 - $400,000 = $100,000 违约放弃的现金流现值（按4%半年期折现）： 第3.5年：$100,000×e-0.04×1 = $100,000×0.9608 = $96,080 第4年：$100,000×e-0.04×2 = $100,000×0.9231 = $92,310 第4.5年：$100,000×e-0.04×3 = $100,000×0.8869 = $88,690 第5年：$100,000×e-0.04×4 = $100,000×0.8521 = $85,210 现值合计 ≈ $96,080 + $92,310 + $88,690 + $85,210 = $362,290 总损失：$50,000 + $362,290 = $412,290
答：金融机构总损失约为 $412,000

📝 第4题 — Biogen Idec期权定价（B-S公式）
假设一个在Nasdaq上市的公司Biogen Idec不分红，股价$50，无风险利率4%，9月8日期权到期还有1年，年波动率30%。 (a) 用B-S公式计算期权价格； (b) 若股价跌至$43.20，波动率40%，重新计算； (c) 即使股价下降-10%，为什么买入期权价格仍上升？ (d) 用Euro Stoxx 50指数期货对冲，指数当前4183点，距到期37天，求期货价格； (e) 基金€7500万，对冲20%，需多少期货合约？
✅ 第4题 解答
(a) B-S公式计算： S=50, X=50, r=4%, σ=30%, T=1年 d1 = [ln(50/50) + (0.04+0.3²/2)×1] / (0.3×1) = (0 + 0.085)/0.3 = 0.2833 d2 = 0.2833 - 0.3×1 = -0.0167 查表：N(0.2833)≈0.6115, N(-0.0167)≈0.4933 c = 50×0.6115 - 50×e-0.04×0.4933 = 30.575 - 50×0.9608×0.4933 = 30.575 - 23.695 = $6.88 (b) 股价$43.20，波动率40%： d1 = [ln(43.2/50)+(0.04+0.4²/2)]/0.4 = (-0.1463+0.12)/0.4 = -0.0658 d2 = -0.0658-0.4 = -0.4658 c = 43.2×N(-0.0658) - 50×e-0.04×N(-0.4658) ≈ $5.90-6.50（具体需查表） (c) 为什么股价下降但期权价格仍上升？ 原因1：波动率从30%上升到40%（危机），波动率越大期权越贵 原因2：隐含波动率上升带来的增值可能超过时间价值损失 原因3：B-S公式中c与σ正相关——波动率上升期权价格上升 (d) Euro Stoxx 50期货价格 F = Se(r-q)T = 4183×e0.04×37/365 = 4183×1.00406 = 4200点 (e) 对冲20%所需合约数 需对冲金额 = €7500万×20% = €1500万 N = 15,000,000 / (4200×10) ≈ 357份（卖空）

📝 第1题（15分）— 利率平价套利
瑞士和美国连续复利的两个月期年利率分别为3%和7%。瑞士法郎即期汇率为$0.6500，两个月远期汇率为$0.6600。问存在什么样的套利机会？
✅ 第1题 解答
步骤1：计算理论远期汇率 根据利率平价理论：F = S × e(rUS-rCH)T = 0.6500 × e(0.07-0.03)×2/12 = 0.6500 × e0.006667 = 0.6500 × 1.00669 = 0.6543 步骤2：比较 市场远期汇率 = 0.6600 理论远期汇率 = 0.6543 市场远期汇率 > 理论远期汇率 → 远期法郎被高估 → 应卖出远期法郎 步骤3：套利策略 ① 借入$1（7%年利率） ② 以即期汇率$0.6500/CHF兑换为 CHF 1.5385 ③ 存入瑞士银行（3%年利率） ④ 同时卖出远期法郎（以0.6600价格） ⑤ 两个月后： 瑞士存款本利和 = 1.5385×e0.03×2/12 = 1.5385×1.00501 = CHF 1.5462 按远期汇率兑换为美元 = 1.5462×0.6600 = $1.0205 偿还美元贷款 = 1×e0.07×2/12 = $1.0117 ⑥ 净利润 = 1.0205 - 1.0117 = $0.0088/每美元
套利利润 = 0.88% 每美元投资

📝 第3题 — 两步二叉树（欧式看涨期权 + 美式看跌期权）
某个股票现价为50美元。连续2个时间步，每步步长3个月，每步股价上涨6%或下跌5%。无风险年利率5%（连续复利）。执行价格51美元。/n(1) 用两步二叉树计算欧式看涨期权价格。/n(2) 计算美式看跌期权价格。

(2) 美式看跌期权 美式看跌期权在每个节点需检查是否提前执行： 节点uu：S=56.18, Puu=max(51-56.18,0)=0，不提前 节点ud：S=50.35, Pud=max(51-50.35,0)=0.65，回推值=0，选择立即执行=0.65 节点du：S=50.35, Pdu=0.65，同上 节点dd：S=45.125, Pdd=max(51-45.125,0)=5.875，回推值可能<5.875，选择立即执行 继续回推并取各节点最大值...

📝 第2题
如果某项资产与利率之间具有很强的正相关关系，那么期货价格与远期价格的关系是？
✅ 答案
期货价格 > 远期价格。因为资产价格与利率正相关→利率上升时资产价格上升→多头盈利→资金可再投资→正的凸性效应→期货价格更高。
📝 第3题
关于期权市场价值的描述？
✅ 答案
期权费 = 内在价值 + 时间价值。实值期权内在价值>0；平值和虚值期权内在价值=0。
📝 第4题
如果期权标的资产为支付红利的股票，期权平价关系应如何调整？
✅ 答案
从当前股价中减去预期红利支付的现值。调整后：c + Xe-rT = p + S - PV(红利)
📝 第5题
下列哪个头寸与标准利率互换多头一方具有相同的利率风险？
✅ 答案
发行浮动利率债券 + 买入固定利率债券。互换多头（收固定付浮动）等价于融资浮动同时投资固定。
📝 第6题
在货币互换中，收入欧元利息支付美元利息，并交换本金。最合适的对冲是？
✅ 答案
相关的外汇远期合约 + 利率互换的组合。
📝 第8题
看跌期权的多头头寸可以由下列哪种方式合成而来？
✅ 答案
看涨期权多头 + 标的资产空头（或卖出期货）。p = c - S + Xe-rT → 看跌=看涨-股票+债券
📝 第9题
金融机构与客户进行一笔普通货币互换交易，收入4.5%利息，支付2%利息，名义本金JPY350000万。互换价值如何变化？

📝 第1题 — 国债期货对冲债券组合
一家基金管理公司计划用国债期货对债券组合进行套期保值。债券组合价值$1亿，4个月后组合久期为4年。目前国债期货价格为122（每$100面值），每张期货合约标的$100,000面值。可交割债券中最便宜债券久期为9年。 (1) 应使用什么期货头寸？合约数量？ (2) 若一个月后CTD债券久期变为7年，对冲者应如何调整？ (3) 若两个月后利率普遍下降但短期利率下降幅度小于长期，对套期保值有何影响？
✅ 解答
(1) 头寸和合约数量 债券组合久期4年，预期利率上升→债券价格下降→应卖出（空头）期货 N = DB×S / (DF×F) = 4×100,000,000 / (9×122,000) = 400,000,000 / 1,098,000 ≈ 364份合约（卖空） (2) 一个月后CTD久期变7年 重新计算所需合约数： N = 4×100,000,000 / (7×122,000) = 400,000,000 / 854,000 ≈ 468份 原已有364份空头，需增加468-364=104份空头合约 (3) 利率曲线变陡的影响 短期利率下降幅度小于长期利率→收益率曲线变陡 长期债券价格上升幅度小于短期债券→组合实际久期变化与预期不同 实际对冲效果会产生偏差，需要动态调整头寸
📝 第2题 — 远期LIBOR计算
1年期LIBOR为10%。一家银行的互换合约规定用12个月的LIBOR与固定利率互换，每年交换一次。2年和3年的互换利率（年复利）分别为11%、12%。求2年和3年的远期LIBOR。
✅ 解答
原理：互换利率是各期限的平价利率，远期利率可由即期利率推算 2年互换利率11% → 第2年LIBOR (1+11%)² = (1+10%)(1+LIBOR2) 1.2321 = 1.1 × (1+LIBOR2) LIBOR2 = 1.2321/1.1 - 1 = 1.12009 - 1 = 12.01%
第2年LIBOR = 12.01%
3年互换利率12% → 第3年LIBOR (1+12%)³ = (1+10%)(1+LIBOR2)(1+LIBOR3) 1.40493 = 1.1 × 1.1201 × (1+LIBOR3) 1+LIBOR3 = 1.40493/(1.1×1.1201) = 1.40493/1.23211 = 1.1403 LIBOR3 = 1.1403 - 1 = 14.03%
第3年LIBOR = 14.03%
📝 第3题 — 养老基金组合风险
某养老基金有€10M资产，全部投资于固定收益证券，平均收益率3.5%。投资组合结构为：75%国债(AAA,久期4.3年) + 25%公司债。利率平行移动125bps，求组合价值变化。
✅ 解答
(1) 组合久期计算 DP = 75%×4.3 + 25%×3.5 = 3.225 + 0.875 = 4.1年 (2) 利率变动对组合价值的影响 ΔP/P ≈ -D × Δy（连续复利近似） Δy = 125bps = 0.0125 ΔP/€10M ≈ -4.1 × 0.0125 = -0.05125 = -5.125% 价值变化 ≈ -€10M × 5.125% = -€512,500 (3) 利率变动时投资者是否获利 如果持有债券到期且不提前卖出，市值变化仅为账面浮亏 实际盈亏取决于投资者是否在利率变动后卖出
📝 第4题 — 两步二叉树（欧式看跌+看涨）
给定股票价格变化路径。S=10，两步二叉树，u=1.188，d=0.84，无风险利率10%。 计算欧式看跌期权和欧式看涨期权的价格。
✅ 解答
已知：S=10, u=1.188, d=0.84, r=10%, Δt=1 p = (e0.1-0.84)/(1.188-0.84) = (1.10517-0.84)/0.348 = 0.762 1-p = 0.238 股价计算： Suu=10×1.188×1.188=14.11 Sud=Sdu=10×1.188×0.84=9.98 Sdd=10×0.84×0.84=7.06 欧式看涨期权（假设X=14）： fuu=max(14.11-14,0)=0.11 fud=max(9.98-14,0)=0 fdd=max(7.06-14,0)=0 fu=e-0.1[0.762×0.11+0.238×0]=0.9048×0.0838=0.0758 fd=e-0.1[0.762×0+0.238×0]=0 f=e-0.1[0.762×0.0758+0.238×0]=0.9048×0.0578=0.0523 欧式看跌期权（假设X=10）： fuu=max(10-14.11,0)=0 fud=max(10-9.98,0)=0.02 fdd=max(10-7.06,0)=2.94 （后续按同样方法回推即可）

📝 第5题 — DAX & STOXX指数期货对冲
一个充分分散的证券投资组合投资于德国和欧洲股票，市值€1000万，预期未来6个月市场下跌。 DAX30当前3712.61点，β=1.2；STOXX50当前2624.96点，β=1.3。 FDAX合约每点€25，当前期货价3783；FESX合约每点€10，当前期货价2667。 (1) DAX总回报指数 vs STOXX50价格指数对期货定价的影响 (2) 分别用两种期货对冲所需合约数 (3) 用ODAX期权Delta对冲初始成本 (4) 构建Delta-Gamma中性组合
✅ 解答
(1) 总回报指数 vs 价格指数 总回报指数包含股利再投资→期货价F=Se(r-q)T q>0 价格指数不包含股利→期货定价需考虑股利影响 STOXX50价格指数的期货价格要加上股利成本的调整 (2) 对冲所需合约数 FDAX（DAX期货）： N = β × S / (F × 每点价值) = 1.2 × 10,000,000 / (3783×25) = 12,000,000 / 94,575 ≈ 127份（卖空） FESX（STOXX期货）： N = 1.3 × 10,000,000 / (2667×10) = 13,000,000 / 26,670 ≈ 487份（卖空） (3) ODAX期权Delta对冲 组合β=1.2 → 等价于DAX的Delta=1.2 用X=3700看跌期权（Δ=-0.415）： 每份看跌期权利率风险 = Δ × 指数 = -0.415 × 3712.61 = -1,540.73 需要对冲的总风险 = β × S × 1% = 1.2 × 10,000,000 = 12,000,000 所需期权数量 = 12,000,000 / (0.415×3712.61) ≈ 7,783份 初始成本 = 7,783 × $296.60 = €2,308,000 (4) Delta-Gamma中性 需要两种期权（如3600看跌和3800看涨） 方程组： Δ × N1 + Δ × N2 + β = 0 Γ × N1 + Γ × N2 = 0 解出N1和N2即可。
